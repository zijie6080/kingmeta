# Platform capability docs

This folder is a read-only reference copy of the platform capability docs for
this agent, refreshed automatically when the platform updates. Browse it to
see what the agent can do — channels, connectors, automations, memory, files,
and more.

Notes:

- Edits here are overwritten on the next refresh; this folder is informational.
- Some capabilities are only available for certain workspaces, plans, or
  channel contexts; the agent reports the live state in chat.
- The agent itself loads these instructions through its own tools, never from
  this folder.

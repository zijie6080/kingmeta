# Conversation Checkpoints

Generated by Base44 Superagent.

Purpose: each conversation gets its own folder containing:
- `memory.md` for facts learned in that conversation.
- `checkpoints.md` for compact summaries of older chat messages.

Checkpoints are not the chat transcript and are not long-term memory. Durable
facts belong in the conversation's `memory.md`; the system loads all
conversation memory files and labels the current conversation separately.

You may edit these files, but the system may rewrite generated checkpoint blocks as conversations continue.
